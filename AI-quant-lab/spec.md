## 3. 指标清单与计算公式

### 3.1 RSI — 相对强弱指标

- **分类**: 动量指标 (oscillator)
- **目的**: 衡量近期涨跌幅的相对力度
- **默认参数**: N = 14 天
- **计算公式**:
  ```
  RS = 平均涨幅(N日) / 平均跌幅(N日)
  RSI = 100 - [100 / (1 + RS)]
  ```
- **信号规则**:
  - RSI > 70 → 超买区（可能回调）
  - RSI < 30 → 超卖区（可能反弹）
  - RSI 与价格走势背离 → 趋势转弱信号
- **代码实现**: `_rsi(close, period=14)`

### 3.2 MACD — 异同移动平均线

- **分类**: 趋势跟随指标
- **目的**: 通过两条指数移动平均线的聚散识别趋势变化
- **默认参数**: (12, 26, 9)
- **计算公式**:
  ```
  DIF 快线 = EMA(close, 12) - EMA(close, 26)
  DEA 慢线 = EMA(DIF, 9)
  MACD 柱  = (DIF - DEA) × 2
  ```
- **信号规则**:
  - DIF 上穿 DEA → 金叉（看涨）
  - DIF 下穿 DEA → 死叉（看跌）
  - DIF 上穿 0 轴 → 多头市场
  - DIF 下穿 0 轴 → 空头市场
- **代码实现**: `_macd(close, fast=12, slow=26, signal=9)`

### 3.3 ATR — 平均真实波幅

- **分类**: 波动率指标
- **目的**: 衡量价格波动的真实幅度
- **默认参数**: N = 14 天
- **计算公式**:
  ```
  TR = max(H - L, |H - PC|, |L - PC|)   // 取三者最大值
  ATR = SMA(TR, 14)
  ```
- **关键用法**:
  - 设置止损: 止损价 = 入场价 ± (ATR × 2~3)
  - 仓位管理: 仓位 = 可承受亏损金额 / (ATR × 乘数)
  - 波动过滤: ATR 收缩 → 变盘前兆
- **代码实现**: `_atr(high, low, close, period=14)`

### 3.4 布林带 — Bollinger Bands

- **分类**: 波动率通道指标
- **目的**: 通过标准差判断价格的相对高低
- **默认参数**: 周期 20, 带宽 2
- **计算公式**:
  ```
  中轨 = SMA(close, 20)
  标准差 σ = std(close, 20)
  上轨 = 中轨 + 2 × σ
  下轨 = 中轨 - 2 × σ
  带宽 %B = (close - 下轨) / (上轨 - 下轨)
  ```
- **信号规则**:
  - 触上轨 → 超买；触下轨 → 超卖
  - 带宽收缩 (squeeze) → 突破前兆
  - 带宽扩张 → 趋势加速
- **代码实现**: `_bollinger(close, period=20, std_dev=2)`
